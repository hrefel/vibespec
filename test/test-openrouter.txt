Build a real-time collaborative task management application with the following features:

- User authentication and authorization with JWT tokens
- Create, edit, and delete tasks with priority levels (low, medium, high, urgent)
- Assign tasks to team members
- Real-time updates when tasks are modified by other users
- Filter tasks by status (todo, in-progress, completed), assignee, and priority
- Dashboard with statistics showing task completion rates and team productivity
- Comment system on tasks for team collaboration
- File attachments for tasks (images, documents)
- Email notifications for task assignments and deadlines
- Mobile-responsive design
- Dark mode support

Tech stack:
- Frontend: React with TypeScript, TailwindCSS, Socket.io-client
- Backend: Node.js, Express, Socket.io, PostgreSQL
- Authentication: JWT with refresh tokens
- File storage: AWS S3 or similar cloud storage

Acceptance criteria:
- Real-time updates should occur within 1 second
- Support at least 100 concurrent users
- Mobile responsive on devices 320px and above
- All forms should have proper validation
- Unit test coverage of at least 80%
